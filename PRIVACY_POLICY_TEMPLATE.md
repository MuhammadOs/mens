## Data Safety Form Tips

When filling out the Data Safety form in Google Play Console, you'll need to declare:

### Data Collection

For each type of data you collect, specify:

- What data is collected
- Why it's collected
- Whether it's shared with third parties
- Whether users can request deletion

### Common Data Types for E-commerce Apps:

- **Account Info**: Email, name, phone (for user accounts)
- **Personal Info**: Name, address (for shipping)
- **Financial Info**: Payment methods (if handling payments)
- **App Activity**: User actions, in-app purchases
- **Device ID**: For analytics or crash reporting

### Third-Party Data Sharing

If you use:

- **Analytics** (e.g., Google Analytics, Firebase): Declare data shared
- **Payment Processors**: Declare financial data shared
- **Cloud Services**: Declare data stored externally
- **Advertising**: Declare advertising IDs and user behavior

Be honest and thorough - Google will review this carefully!
